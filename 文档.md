## 4. 详细检测项（专业网络测试标准）
### 4.1 基础网络检测
1. 本机网卡状态：是否启用、IP地址获取正常
2. 网关连通性：ping网关，验证内网通畅
3. 公网连通性：ping公共DNS（223.5.5.5/8.8.8.8），验证外网通畅

### 4.2 DNS检测（解决网站/API解析失败）
1. 目标域名解析（github.com/业务API域名）
2. 解析速度测试
3. 公共DNS可用性（阿里/腾讯/谷歌）
4. 本地DNS缓存状态

### 4.3 端口&代理检测（**你的核心故障点**）
1. **指定端口监听检测**：检测10808/7890等代理端口是否有服务运行（解决：配置了代理但软件没启动）
2. **目标端口连通性**：检测github.com:443、API端口是否可连接（解决：清除代理后443不通）
3. **系统代理检测**：Windows/Mac系统全局代理配置
4. **代理有效性验证**：测试配置的代理是否能正常转发请求

### 4.4 Git专项检测（**定制化解决你的Git问题**）
1. Git全局配置检测：`http.proxy`/`https.proxy`（排查无效代理）
2. Git本地仓库配置检测：是否存在局部代理覆盖
3. Git远程仓库连通性：`git pull`/端口探测
4. Git SSL验证状态：是否开启SSL导致连接失败
5. Git远程地址有效性：是否为官方地址/镜像地址

### 4.5 路由&SSL检测
1. 路由追踪（tracert）：定位网络超时节点
2. HTTPS证书验证：SSL证书是否有效
3. 防火墙出站规则：443端口是否被屏蔽

## 5. 智能自动修复策略（**针对你的故障量身定制**）
| 检测到的故障 | 自动修复方案 | 优先级 |
|--------------|--------------|--------|
| Git配置了**无效代理**（端口无监听） | 一键清除Git全局/本地代理配置 | 最高 |
| github.com 443端口不通 | 1. 临时禁用Git SSL验证 2. 配置Git国内镜像 | 高 |
| DNS解析异常 | 自动切换为阿里DNS(223.5.5.5)/腾讯DNS | 高 |
| 系统代理与Git代理冲突 | 清除系统代理+Git代理 | 中 |
| Git配置错误 | 重置Git全局网络配置 | 中 |
| 本地DNS缓存污染 | 刷新DNS缓存 | 中 |

## 7. 故障分级标准（专业网络测试）
- **致命**：无公网、网卡禁用 → 无法使用网络
- **严重**：Git端口不通、无效代理、DNS解析失败 → 核心业务不可用
- **警告**：DNS慢、路由超时、代理配置冗余 → 网络不稳定
- **正常**：所有项通过

## 8. 完整Python工具代码（可直接运行/打包）
```python
# -*- coding: utf-8 -*-
"""
网络&Git一体化诊断修复工具
适配：Windows/Mac/Linux
核心解决：代理失效、Git 443端口不通、DNS异常、网络受限
"""
import socket
import subprocess
import sys
import os
from colorama import init, Fore, Style

# 初始化彩色日志
init(autoreset=True)

# 配置项
TEST_DOMAINS = ["github.com", "www.baidu.com"]
TEST_PORTS = [443]
TEST_DNS = ["223.5.5.5", "8.8.8.8"]
GIT_PROXY_KEYS = ["http.proxy", "https.proxy"]


class NetworkGitTool:
    def __init__(self):
        self.report = []
        self.platform = sys.platform

    # 工具打印
    def log(self, level, msg):
        color = Fore.GREEN
        if level == "WARN":
            color = Fore.YELLOW
        elif level == "ERROR":
            color = Fore.RED
        elif level == "INFO":
            color = Fore.CYAN
        print(f"{color}{msg}{Style.RESET_ALL}")
        self.report.append(f"[{level}] {msg}")

    # 1. ping检测
    def ping_test(self, host):
        try:
            cmd = f"ping -n 2 {host}" if self.platform == "win32" else f"ping -c 2 {host}"
            res = subprocess.run(cmd, shell=True, stdout=subprocess.PIPE, stderr=subprocess.PIPE, timeout=10)
            return res.returncode == 0
        except:
            return False

    # 2. 端口监听/连通性检测
    def port_test(self, host, port):
        try:
            sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
            sock.settimeout(3)
            result = sock.connect_ex((host, port))
            sock.close()
            return result == 0
        except:
            return False

    # 3. DNS解析检测
    def dns_test(self, domain):
        try:
            ip = socket.gethostbyname(domain)
            return True, ip
        except:
            return False, None

    # 4. Git配置检测
    def git_config_check(self):
        proxy_config = {}
        try:
            for key in GIT_PROXY_KEYS:
                res = subprocess.run(f"git config --global --get {key}", shell=True, stdout=subprocess.PIPE,
                                    stderr=subprocess.PIPE, text=True)
                val = res.stdout.strip()
                if val:
                    proxy_config[key] = val
            return proxy_config
        except:
            return {}

    # 5. 清除Git无效代理
    def fix_git_proxy(self):
        self.log("INFO", "🔧 开始修复：清除Git全局无效代理")
        try:
            for key in GIT_PROXY_KEYS:
                subprocess.run(f"git config --global --unset {key}", shell=True, stdout=subprocess.PIPE,
                              stderr=subprocess.PIPE)
            self.log("INFO", "✅ Git代理已清除完成")
        except Exception as e:
            self.log("ERROR", f"❌ 修复失败：{str(e)}")

    # 6. 修复Git SSL验证（临时解决443连接失败）
    def fix_git_ssl(self):
        self.log("INFO", "🔧 开始修复：禁用Git SSL验证（测试用）")
        subprocess.run("git config --global http.sslVerify false", shell=True, stdout=subprocess.PIPE,
                       stderr=subprocess.PIPE)
        self.log("INFO", "✅ Git SSL验证已禁用")

    # 7. 刷新DNS
    def fix_dns(self):
        self.log("INFO", "🔧 开始修复：刷新本地DNS缓存")
        cmd = "ipconfig /flushdns" if self.platform == "win32" else "sudo dscacheutil -flushcache"
        subprocess.run(cmd, shell=True, stdout=subprocess.PIPE, stderr=subprocess.PIPE)
        self.log("INFO", "✅ DNS缓存已刷新")

    # 全量检测
    def full_check(self):
        self.log("INFO", "=" * 50)
        self.log("INFO", "🚀 网络&Git诊断修复工具 - 全量检测开始")
        self.log("INFO", "=" * 50)

        # 1. 公网连通性
        self.log("INFO", "\n1. 基础网络检测...")
        ping_ok = self.ping_test("223.5.5.5")
        if ping_ok:
            self.log("OK", "✅ 公网连通正常")
        else:
            self.log("ERROR", "❌ 公网连接失败，请检查网络")

        # 2. DNS检测
        self.log("INFO", "\n2. DNS解析检测...")
        for domain in TEST_DOMAINS:
            dns_ok, ip = self.dns_test(domain)
            if dns_ok:
                self.log("OK", f"✅ {domain} 解析成功：{ip}")
            else:
                self.log("ERROR", f"❌ {domain} 解析失败")
                self.fix_dns()

        # 3. 端口检测（核心：github 443）
        self.log("INFO", "\n3. 端口连通性检测...")
        for domain in TEST_DOMAINS:
            for port in TEST_PORTS:
                port_ok = self.port_test(domain, port)
                if port_ok:
                    self.log("OK", f"✅ {domain}:{port} 端口连通正常")
                else:
                    self.log("ERROR", f"❌ {domain}:{port} 端口连接失败（防火墙/网络限制）")

        # 4. Git代理检测（你的核心问题）
        self.log("INFO", "\n4. Git配置检测...")
        git_proxy = self.git_config_check()
        if git_proxy:
            self.log("WARN", f"⚠️ 检测到Git代理配置：{git_proxy}")
            self.log("WARN", "⚠️ 可能为无效代理（代理软件未启动）")
        else:
            self.log("OK", "✅ Git无代理配置")

        # 5. 代理端口检测（10808）
        self.log("INFO", "\n5. 代理端口监听检测...")
        proxy_port_ok = self.port_test("127.0.0.1", 10808)
        if proxy_port_ok:
            self.log("OK", "✅ 10808代理端口正常监听")
        else:
            self.log("ERROR", "❌ 10808端口无监听（代理软件未启动）")

        self.log("INFO", "\n" + "=" * 50)
        self.log("INFO", "🏁 全量检测完成")
        self.log("INFO", "=" * 50)

    # 一键修复
    def auto_fix(self):
        self.log("INFO", "\n" + "=" * 50)
        self.log("INFO", "🔧 自动修复开始")
        self.log("INFO", "=" * 50)

        # 修复1：清除Git无效代理
        self.fix_git_proxy()
        # 修复2：刷新DNS
        self.fix_dns()
        # 修复3：禁用Git SSL
        self.fix_git_ssl()

        self.log("INFO", "\n" + "=" * 50)
        self.log("INFO", "✅ 自动修复完成！重新检测验证效果")
        self.log("INFO", "=" * 50)

    # 导出报告
    def save_report(self):
        with open("网络检测报告.txt", "w", encoding="utf-8") as f:
            f.write("\n".join(self.report))
        self.log("INFO", "\n📄 报告已保存：网络检测报告.txt")


# 主程序
if __name__ == "__main__":
    tool = NetworkGitTool()

    while True:
        print("\n" + "=" * 50)
        print("网络&Git诊断修复工具")
        print("1. 一键全量检测")
        print("2. 一键自动修复")
        print("3. 保存检测报告")
        print("0. 退出工具")
        print("=" * 50)
        choice = input("请输入操作序号：")

        if choice == "1":
            tool.full_check()
        elif choice == "2":
            tool.auto_fix()
        elif choice == "3":
            tool.save_report()
        elif choice == "0":
            tool.log("INFO", "👋 工具退出")
            break
        else:
            tool.log("WARN", "⚠️ 输入无效，请重新选择")
```
